## 🚀 Getting Started

1. **Clone the repository**:
```bash
git clone https://github.com/yourusername/stews-guide-to-tbilisi.git
cd stews-guide-to-tbilisi
```

2. **Install dependencies**:
```bash
npm install
```

3. **Set up environment variables**:
```bash
cp .env.example .env.local
# Add your AI service API keys
```

4. **Run the development server**:
```bash
npm run dev
```

5. **Open your browser**:
Navigate to [http://localhost:3000](http://localhost:3000) and experience the AI-powered guide!
